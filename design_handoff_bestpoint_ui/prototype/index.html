flujo inicial
Onboarding con cards despegables → feed con header de usuario, scroll horizontal + vertical de posts, y una barra inferior dinámica que persiste los últimos 4 atajos usados. El botón + abre un buscador flotante con todas las acciones (crear nota, tarea, idea, agenda, reunión, encuesta, post, buscar en posts, editar, eliminar, reportar avance, comentar).

📱 Vista móvil principal

Prototipo interactivo. Navegá con los tabs, ejecutá herramientas desde la barra inferior o el buscador y mirá cómo se reordena MRU.

🖥️ Vista web mockup centrado

La web muestra la misma app centrada dentro de un viewport oscuro con cromo de navegador — consistente con el DNA móvil.
